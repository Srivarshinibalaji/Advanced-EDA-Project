**Advanced Exploratory Data Analysis (EDA) Project**

**Project Overview**

This project focuses on performing Advanced Exploratory Data Analysis (EDA) using Python.  
The main goal is to clean the dataset, handle missing values, detect and remove outliers, and apply feature engineering to prepare the data for further machine learning tasks.

**Objectives**

Understand dataset structure  
Perform data cleaning  
Handle missing values  
Detect and remove outliers  
Create new features (feature engineering)  
Visualize data insights  

**Tools and Technologies Used**

Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  

**Project Files**

main.py → Python script containing full EDA process  
Dataset.csv → Original dataset  
cleaned_dataset.csv → Processed dataset after cleaning  
graphs/ → Folder containing all generated visualizations  

**Steps Performed**

1. Data Loading  
The dataset was loaded using Pandas for analysis.

2. Data Cleaning  
Handled missing values  
Removed duplicate entries  
Corrected data types where necessary  

3. Exploratory Data Analysis  
Statistical summary of data  
Understanding distributions of variables  

4. Outlier Detection  
Used IQR method to detect outliers  
Removed or treated extreme values  

5. Feature Engineering  
Created new meaningful features from existing data  

6. Data Visualization  
Generated graphs using Matplotlib and Seaborn  
Correlation matrix and distribution plots created  

**Output Generated**

Cleaned dataset saved as cleaned_dataset.csv  
Visualizations saved inside graphs folder  
Correlation matrix and boxplots generated successfully  

**Conclusion**

This project demonstrates the complete EDA pipeline including data cleaning, transformation, visualization, and feature engineering.  
The dataset is now ready for machine learning model development.
